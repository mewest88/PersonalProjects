The RCStreamCypher assignment consists of only the main.cpp file.

To test, there is an encrpyt() method and a decrypt() method.

Encrypt(std::string pass, std::string message) takes a string password and a string message.

The decrypt the passed in message, run decrypt(std::string pass, std::string message), passing the same arguments as for decrypt.

There are some example tests at the bottom of the main that an be used as examples.
